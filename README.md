# 🌾 KrishiRaksha

## AI-Powered Agricultural Crisis Response Platform

KrishiRaksha is a Flutter + Firebase powered solution built under the **Rapid Crisis Response** track for **Google Solution Challenge 2026**.

The platform helps farmers and local traders respond to agricultural crises caused by:

- Market volatility  
- Climate uncertainty  
- Post-harvest losses

Using predictive intelligence, real-time alerts and AI assistance, KrishiRaksha helps users **Predict, Prevent and Respond** to crises.

---

# 🚨 Problem Statement

Farmers frequently face losses due to:
- Sudden crop price fluctuations  
- Unpredictable rainfall and sowing conditions  
- Crop spoilage after harvest  
- Lack of real-time market intelligence

KrishiRaksha addresses these challenges through an integrated agricultural crisis response system.

# ⚙️ Setup

## Clone repository
```bash
git clone https://github.com/yourusername/Krishi_Raksha.git
cd Krishi_Raksha
```

---

## Install Flutter dependencies
```bash
flutter pub get
```

---

## Run application
```bash
flutter run
```

---

## Firebase Functions setup
```bash
cd functions
npm install
firebase deploy --only functions
```
